ЗАДАЧА:
Верни строго JSON со следующей структурой (никакого markdown, никаких пояснений):
{
  "direction": "BUY" | "SELL" | "NEUTRAL",
  "magnitude": float ∈ [0.0, 1.0],
  "affected_tickers": [список тикеров из MOEX, для которых это релевантно],
  "horizon_min": int (через сколько минут реакция рынка должна произойти),
  "reason": "Краткое объяснение на русском, до 200 символов. Конкретно: ЧТО двигает цену и КАК."
}

ПРАВИЛА:
- direction = NEUTRAL если новость неоднозначна или уже учтена рынком
- magnitude > 0.7 только для очень сильных катализаторов (санкции, отмена дивидендов, M&A)
- horizon_min: 15-30 для микроновостей, 60-120 для среднесрочных
- affected_tickers — только из списка: LKOH, SBER, ROSN, GAZP, VTBR, YDEX, PLZL, T, NVTK, X5, GMKN, MGNT, ALRS, AFLT, CHMF, NLMK, MOEX, SNGSP, MTSS, PIKK
